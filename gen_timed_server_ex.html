Sample implementation of a gen_timed_server.

Version: $Revision$ $Date$

Behaviours: gen_timed_server.

Authors: Serge Aleynikov (saleyn@gmail.com).

Description

Sample implementation of a gen_timed_server. Implements a server that executes for a given number of milliseconds and then exits to be rescheduled in accordance with provided supervision options.
